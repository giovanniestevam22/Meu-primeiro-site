introdução ao html

HTML significa Hyper Text Markup Language

HTML é a linguagem de marcação padrão para criação de páginas da Web HTML descreve a estrutura de uma página da Web HTML consiste em uma série de elementos Os elementos HTML informam ao navegador como exibir o conteúdo Os elementos HTML rotulam partes de conteúdo como "este é um título", "este é um parágrafo", "este é um link", etc.
